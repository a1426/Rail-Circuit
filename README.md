# Rail Circuit Converter
## Purpose:
This program is intended to generate a Qiskit model of a quantum circuit from the corresponding rail diagram using Artificial Intelligence. 

## Installation:
1. Clone the repo
   ```
   git clone https://github.com/a1426/Rail-Circuit-Converter
   ```

## Usage
Pass in the path of the image to the ```main()``` function, located in ```main.py```, which will return a ```QuantumCircuit``` object.
